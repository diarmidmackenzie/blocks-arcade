A-Frame VR Tetris Games - Pre-alpha

What is this?

Pre-alpha development versions of some VR Tetris games. These are at a very early stage of development - they are much closer to a proof of concept than a full game at this stage.

However, they are playable (give or take a couple of bugs), and quite fun! I am more than happy for anyone who wants to play these games, and share them with others too. But if hitting bugs and glitches is going to make you sad, I suggest you wait a few weeks, when hopefully the quality will be a bit better.

Also, please be aware I haven't done any work yet on performance, to ensure high frame rates. So if you are sensitive to this sort of thing, please take care.

I've been testing this on Oculus Quest 2, using the Oculus Browser. In theory it should work with other Web XR compatible devices. If you try it on a different device, and it works (or doesn't work), I'd be interested to know. But I probably won't be fixing it up to work on additional systems any time soon.

If you don't have a VR headset, you can play it on a laptop or dektop PC, using keyboard controls, but sadly it falls a long way short of the VR experience.

Game modes available today:

2D Tetris - The classic game. Not very different in 3D, except you can play from either side of the board.

3D Tetris - A 3D version of Tetris. Difficult on a 2D browser, but much easier in 3D. You have to fill an entire 5x5 layer before it disappears.

2D "Pentris" - Like classic 2D Tetris, but with all possible 5-block tiles. This is much harder! But with a bit of practice and familiarity with the new shapes, it's quite playable.

Space Balls! - It's just 2D Tetris again, with a different skin...

Controls

6DOF controls: Grip Down + move to move, Trigger Down + rotate to rotate. Grip+Trigger together will allow moving and rotation together. No control yet to drop blocks.

My view on the 6DOF controls is that they are OK, but not great, and I'm looking to make some changes in the near future...

Keyboard controls (for computer browser):

  • for 2D games: Z/X left/right, RShift/Enter to rotate, Space to drop.
  • for 3D games: YGHJ for movement in the plane, NumPad 4-9 for rotations.
The default position of the board is optimized for VR. On desktop, take 1-2 steps back (press S, or down-arrow, 1-2 times) to see the whole playing arena.

The scoring is: 1 point for a line/layer, N-squared points for N lines/layers at a time.

There is no "start game" or "restart game" control yet. Just hit Refresh to restart. If anything else goes wrong or weird, I also recommend you exit VR, hit Refresh in the browser, and then re-enter VR.

Other info

Some technical notes: All game modes use the same Javascript Engine, and the differences amount to a few lines of HTML.

My hope is to make many more aspects of the game customizable using HTML only. - block colors, sizes, textures, controls, scoring etc. etc.
